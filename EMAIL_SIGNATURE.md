## How to install in Gmail (2 minutes)

1. Open Gmail → click the **gear icon** (top right) → **See all settings**
2. Scroll down to **Signature** section
3. If you have an existing signature, edit it. If not, click **+ Create new**
4. Paste either version above into the editor
5. (Optional but recommended) Set:
   - **Signature defaults** → use this signature for new emails AND replies
   - **"Insert signature before quoted text"** ✅ checked
6. Scroll to bottom → click **Save changes**

## How to install in Apple Mail (1 minute)

1. Mail → Settings → **Signatures** tab
2. Pick the email account (left column)
3. Click **+** to add new, or edit existing
4. **Important:** uncheck "Always match my default message font" so the colors/styling survive
5. Paste either version above

## How to install in Outlook

1. File → Options → **Mail** → **Signatures…**
2. Click **New** or edit existing
3. Paste either version
4. Set as default for "New messages" and "Replies/forwards"
